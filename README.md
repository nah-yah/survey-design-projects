## Français

### À propos de ce portfolio

Ce dépôt rassemble une série de **questionnaires d'enquête** conçus dans le cadre de la construction de mon portfolio. Chaque projet présente un instrument de collecte complet, structuré selon les standards **XLSForm** et déployable sur les plateformes **ONA**, **KoBoToolbox** ou **Enketo**.

Les questionnaires couvrent une **diversité de thématiques et de contextes** : études de marché, satisfaction client, évaluation de programmes, recherche académique, suivi de projets, santé publique, éducation, environnement, ressources humaines, etc. Les terrains d'application sont variés et ne se limitent à aucun secteur ou pays en particulier.

### Structure du dépôt

Chaque projet est organisé dans son propre dossier
```text
00X_projet/
├── questionnaire.xlsx       # Formulaire XLSForm
├── note_methodologique.pdf  # Contexte, objectifs, plan de sondage
└── README.md                # Documentation du projet
```

### Standards techniques

- **Format** : XLSForm (compatible ODK / KoBo / ONA)
- **Langues** : Français, anglais, créole, ou versions multilingues selon le projet
- **Validation** : Contraintes (`constraint`), sauts conditionnels (`relevant`), choix dynamiques, calculs internes
- **Déploiement** : Testé sur Enketo Web et application mobile ONA Collect

### Ce que vous trouverez

Pour chaque enquête : objectifs, population cible, modules thématiques, durée moyenne d'administration, mode de collecte recommandé, et le formulaire prêt à l'emploi.

### Note importante

Les questionnaires publiés ici sont **fictifs** à des fins de démonstration.

## English

### About this portfolio

This repository gathers a series of **survey questionnaires** designed as part of my work of building my portfolio. Each project showcases a complete data collection instrument, structured according to **XLSForm** standards and ready to deploy on **ONA**, **KoBoToolbox**, or **Enketo**.

The questionnaires cover a **wide range of topics and contexts**: market research, customer satisfaction, program evaluation, academic research, project monitoring, public health, education, environment, human resources, and more. Applications are varied and not limited to any specific sector or country.

### Repository structure

Each project is organized in its own folder:
```text
00X_project/
├── questionnaire.xlsx # XLSForm
├── methodological_note.pdf # Context, objectives, sampling plan
└── README.md # Project documentation
```

### Technical standards

- **Format** : XLSForm (compatible with ODK / KoBo / ONA)
- **Languages** : French, English, Creole, or multilingual versions depending on the project
- **Validation** : Constraints, skip logic, dynamic choices, internal calculations
- **Deployment** : Tested on Enketo Web and ONA Collect mobile app

### What you'll find

For each survey: objectives, target population, thematic modules, average administration time, recommended collection mode, and the ready-to-use form.

### Sou pòtfolyo sa a

Repozitwa sa a rasanble yon seri **kesyonè ankèt** mwen konseptyalize pou pòtfolyo mwen. Chak pwojè prezante yon zouti kolekt done konplè, ki estriktire dapre estanda **XLSForm** epi ki ka deplwaye sou platfòm **ONA**, **KoBoToolbox**, oswa **Enketo**.

Kesyonè yo kouvri yon **gran varyete tèm ak kontèks** : etid mache, satisfaksyon kliyan, evalyasyon pwogram, rechèch akademik, swivi pwojè, sante piblik, edikasyon, anviwònman, resous imèn, elatriye. Aplikasyon yo varye epi yo pa limite ak okenn sektè oswa peyi an patikilye.

### Estrikti repozitwa a

Chak pwojè òganize nan pwòp dosye li :
```text
00X_pwojè/
├── kesyone.xlsx # Fòmilè XLSForm
├── nòt_metodolojik.pdf # Kontèks, objektif, plan echantiyonaj
└── README.md # Dokimantasyon pwojè a
```


### Estanda teknik

- **Fòma** : XLSForm (konpatib ak ODK / KoBo / ONA)
- **Lang** : Fransè, anglè, kreyòl, oswa vèsyon miltileng selon pwojè a
- **Validasyon** : Kontrent, sòt kondisyonèl, chwa dinamik, kalkil entèn
- **Deplwaman** : Teste sou Enketo Web ak aplikasyon mobil ONA Collect

### Sa w ap jwenn
